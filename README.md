# ws-test-generator
Generates test classes for WebSocket APIs. 

### Prerequisites
1. Node: https://nodejs.org/en/download/
1. Docker: https://docs.docker.com/get-docker/

## Run the app
1. In the terminal, `cd` into directory
1. `$ npm install`
1. `$ npm run start`
1. Visit `http://0.0.0.0:8080` on your browser. Ta-da!
